# Explainable AI for Cardiovascular Risk Assessment

## Overview

This project predicts cardiovascular disease risk using Machine Learning and explains the model's predictions using SHAP (SHapley Additive Explanations).

The notebook covers the complete machine learning workflow, including data preprocessing, model training, evaluation, and explainable AI techniques.

---

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Random Forest Classifier
- Logistic Regression comparison
- Model evaluation
- SHAP Explainability
- Feature importance visualization

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- SHAP
- Jupyter Notebook

---

## Dataset

- UCI Heart Disease Dataset

1. Load Dataset
2. Data Cleaning
3. Feature Engineering

## How to Run

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Explainable-AI-Cardiovascular-Risk-Assessment.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
Explainable_AI_for_Cardiovascular_Risk_Assessment.ipynb
```

and run all cells.
