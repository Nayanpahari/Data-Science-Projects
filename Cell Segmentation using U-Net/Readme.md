# Cell Segmentation using U-Net

## Overview
This project performs automated cell/nuclei segmentation on microscopy images using a U-Net deep learning architecture.

## Features
- Image preprocessing pipeline
- U-Net based segmentation
- Model training & evaluation
- Prediction visualization

## Tech Stack
- PyTorch
- OpenCV
- NumPy

## Results
- Achieved segmentation with visually accurate masks
- Evaluated using Dice Score and IoU
